Example 1. Editing image by <img> tag

If you click the image on this demo Image Editor is opened.

When you save result image it is being uploaded to the server with updating the source image.

Click on image to edit it

Example 2. Editing image by URL

Image specified in URL textfield will be edited when you click a button.

After you save an image it will be shown as preview, textfield URL will be also updated.

So if you want to edit external image in any case you will get link to your own website.

Note. If you paste external URL here, the external server must have correctly configured CORS policy.
